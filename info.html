HTML5 Introduction

HTML stands for HyperText markup language

  • HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of a Web page.
  • HTML consists of a series of elements.

Features of HTML5:

  • Create a web page using tags.
  • Easy DOCTYPE declaration i.e. < !DOCTYPE html>
  • Use graphics and display text in different fonts, sizes and color.
  • Enrich semantic content by including < header> < footer> < article> < section> and < figure> are added
  • Create hyperlinks to navigate to different documents present on the web.
  • It has introduced new multimedia features which support audio and video controls by using < audio> and < video> tags.
